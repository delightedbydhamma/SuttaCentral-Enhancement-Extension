# Development

## Start server

### Chrome

```
pnpm dev
```

### FireFox

```
pnpm dev -b firefox
```

## Create package

### Chrome

```
pnpm zip
```

### Firefox

```
pnpm zip -b firefox
```

zipped packages can be found in `.output`
